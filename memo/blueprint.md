# Discord Bot を作りたい

## 機能
discordの会話上で出た技術用語を、必要に応じて簡単に説明してくれるBOTを作りたい

### 例
@BOT名 BFS

```
BFS【びーえふえす】

Breadth First Search, 幅優先探索

探索アルゴリズムのひとつ。グラフ構造などにおいて、始点からの距離が最も近い箇所から順番に探索していく手法。主にキューが使用される。
```

## 設計方針・参考資料
- BOTに単語名を聞くことで、意味を簡単に教えてくれる機能
    - 既に意味が登録されている場合、それを出力。でなければ、Geminiなどで緊急生成
- チャンネルにおける直近5件程度のメッセージについて、(或いは返信が読み取れるのであれば該当するメッセージ)説明できそうな単語を検知する機能
- 意味の登録・編集・削除ができる

### 参考資料
人狼ジャッジメントのチャットにて、CO, 黒, 吊るといった単語が出現すると、それを感知して意味が知れるボタンが出現する機能があったような。それが参考になりそう。

### BOTの性格
MCC内のみでの運用を想定するため、別に無機質なBOTでなくてもよさそう。万が一、全世界に公開することになったら、改めて考え直すべきか。
ただ、コスト面から考えても、性格は単一にした方が良さそう。以下のどれかが良さそう。

## 名前候補
- dictiord(ディクソード)

    - 書きにくく読みにくい
- jishocord(ジショコード)
    - 辞書 + discord
- TechWord(テックワード)
    - 技術系の鯖運用に限る
- Wikiちゃん(うぃきちゃん)

## データ
dictionary
- word: string
単語
- fullWord: srting | null
略称の場合正式名称
- Japanese: string | null
正式名称が英語の場合日本語の意味
- summary: string
概要。一文で簡潔に
- detail: string
詳細。2~3文で。

inconsistent(表記ゆれ)
- word: string
単語
- fix: string
dictionaryに含まれている単語。fixに直してから探索。
